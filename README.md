# SIR Parameter Estimation (Toy Model)

This project demonstrates parameter estimation for an SIR model using:

- Least Squares
- Maximum Likelihood Estimation (MLE)
- Bayesian Inference

## Structure

- `src/`: model and estimation functions
- `notebooks/`: experiments and analysis
- `data/`: synthetic dataset

## Methods

1. Generate synthetic data
2. Estimate parameters using LS
3. Estimate parameters using MLE
4. Compute posterior distribution (Bayesian)

## Goal

Understand differences between:
- LS
- MLE
- MAP
- Posterior distribution
